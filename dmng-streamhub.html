DMNG StreamHub

Réalisation de l'application web temps réel DMNG StreamHub

Le DMNG StreamHub est l'application de réception du système DMNG compatible avec toutes les solutions de transmission de live vidéo de la société AVIWEST : ​​DMNG série PRO, IBIS DMNG et l'App DMNG (http://www.aviwest.com/product/dmng-streamhub/).

La réalisation de l'application a nécessité le choix d'une architecture web faisant appel au plus récentes avancées en terme de développements web, avec notamment l'utilisation de la pate-forme Node.js pour répondre aux contraintes de temps réel.

Environnement technique

  • Node.js
  • Socket.IO
  • Framework applicatif Express
  • HTML dynamique et MVC avec Angular.js
  • Framework CSS Bootstrap
